## Decisions

- **`approvalProgress` as JSONB on `agent_blueprints`** ‚Äî no new table; all read paths already load this row; append-only semantics preserved.
- **Policy versioning via new row (not UPDATE)** ‚Äî preserves historical `evaluatedPolicyIds` integrity (ADR-003 no-hard-delete); superseded rows never deleted, always fetchable by UUID.
- **Legacy path when `approvalChain = []`** ‚Äî backward compat is unconditional; no migration needed for existing `in_review` blueprints.
- **SOD check extended to all prior step approvers** ‚Äî `createdBy` check alone is insufficient for multi-person chains; all `approvalProgress[*].approvedBy` checked against `userEmail`.
- **`AnyPgColumn` return type annotation** ‚Äî correct Drizzle-ORM pattern for self-referential FKs; replaces eslint-disable comment from previous session.
