# Session 030 — 2026-03-14

## Summary

Phase 26: AgentCore Phase 2 (Direct Deploy) — Intellios now calls the Amazon Bedrock Agent API directly from a reviewer's single click. An approved blueprint is translated to a Bedrock `CreateAgent` + `CreateAgentActionGroup` + `PrepareAgent` call sequence; the session polls until the agent reaches `PREPARED` status (max 30s); success stores the `AgentCoreDeploymentRecord` in `agent_blueprints.deployment_metadata`, promotes status to `deployed`, and writes two audit entries (`blueprint.agentcore_deployed` + `blueprint.status_changed`). All existing webhooks and notifications fire automatically via the event bus. AWS credentials are never stored in the database. Enterprise AgentCore config (region, IAM role ARN, model, optional guardrail) lives in Admin Settings → Deployment Targets.
