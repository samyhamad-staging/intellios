## Decisions

- **Token length**: 32 bytes hex (64 chars) — matches existing pattern in user-invitations system
- **AI orchestrator model**: `claude-haiku-4-5-20251001` — fast + cost-efficient for background fire-and-forget
- **Collaboration synthesis visibility**: stakeholders see synthesized insights (not raw contributions) — privacy-preserving
- **RACI interview tone**: `accountable` = final decision authority → constraints/non-negotiables; `responsible` = delivery owner → requirements/implementation; `consulted` = domain expert → domain-specific; `informed` = awareness → concerns/dependencies
- **Auto-create invitation on insight approval**: when a designer approves a "suggestion" insight with `action === "invite"`, the system auto-creates the invitation and sends the email — closes the orchestrator→action loop
