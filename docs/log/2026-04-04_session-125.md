### W3-09: Optimistic Updates

**Investigation:**

- Read `src/components/registry/lifecycle-controls.tsx` — `handleTransition` uses `setTransitioning(true)` → PATCH → `onStatusChange(next)` on success. No optimism: button disables, spinner shows, UI updates only after round-trip.
- Read `src/app/monitor/page.tsx` — `handleCheckOne` posts to `/api/monitor/${agentId}/check` then calls `invalidateQueries`. Row shows `checkingId === agent.agentId` to dim button. No cache optimism.
- Read `src/app/registry/workflow/[id]/page.tsx` — `transition()` function PATCHes status, updates local `useState` on success only.
- Confirmed `useQueryClient` is available in all three files (all wrapped in `QueryClientProvider` via root `Providers`).
- `review/page.tsx` "Assign to me" and `monitor/page.tsx` "Acknowledge" are both localStorage-only — already instant, no server mutation.

**Design decision:** Use React Query `useMutation` pattern — `onMutate` snapshots + patches cache, `onError` restores snapshot, `onSettled` background-refetches. `transitioning` / `checkingId` state replaced by `mutation.isPending` + `mutation.variables`.

**Changes:**

**`src/components/registry/lifecycle-controls.tsx`:**
1. Added `useQueryClient, useMutation` imports from `@tanstack/react-query`.
2. Added `RegistryEntry` import from `@/lib/query/fetchers` (for typed cache patch).
3. Added `queryKeys` import from `@/lib/query/keys`.
4. Removed `transitioning` state — replaced by `statusMutation.isPending`.
5. `handleTransition` replaced with `statusMutation` using:
   - `onMutate`: cancel in-flight registry agents query, snapshot previous data, patch cache entry for `agentId` to `nextStatus`, call `onStatusChange(nextStatus)` immediately.

   - `onSettled`: `invalidateQueries({ queryKey: queryKeys.registry.agents() })` to reconcile.

7. Error state reads `statusMutation.error?.message`.

**`src/app/registry/workflow/[id]/page.tsx`:**

    - `onError`: restore `previousWorkflow` via `setWorkflow(context.previousWorkflow)`, restore workflows cache snapshot.
