### Landing page rewrite

**File:** `src/app/landing/page.tsx`

**Before:** 209-line placeholder with basic nav, hero text, 5-step pipeline cards, 3 differentiator cards, 4 role cards, footer CTA. No icons, no gradient, flat styling.

**After:** Full premium marketing page (~320 lines):

| Section | Implementation |
|---|---|
| **Sticky nav** | Logo (indigo bolt icon + wordmark), 4 nav links, Sign in text link + "Request access" indigo pill button with backdrop-blur/bg-white/90 |
| **Hero** | Large indigo gradient blob (absolute positioned, `blur-3xl`, `opacity-20`), badge pill with `ChevronRightIcon`, 4xl→6xl headline with `text-indigo-600` accent span, subtext, dual CTAs (indigo filled + ghost bordered), social proof line |
| **Compliance badges** | `SR 11-7`, `EU AI Act`, `NIST AI RMF`, `ISO 42001`, `SOC 2 Type II` — color-coded ring-1 badge pills on gray-50 background |
| **Pipeline** | 5 steps in centered icon circles (Heroicons), connecting gradient line on desktop, numbered `01–05` in monospace indigo, responsive `grid-cols-1 sm:grid-cols-3 lg:grid-cols-5` |
| **Primary features** | 3 cards with colored icon containers (indigo/violet/emerald), rounded-2xl border, hover shadow transition |
| **Stats** | 4 metrics (`5-step`, `3`, `100%`, `Zero`) in 4xl indigo-600, `grid-cols-2 lg:grid-cols-4` |
| **Secondary features** | 8-item grid with indigo-600 icon boxes: separation-of-duties, policy library, posture monitoring, white-label, MRM exports, SSO/RBAC, blueprint history, behavioral test evidence |
| **Roles** | 4 role cards (Architect/Reviewer/Compliance Officer/Admin) with icon + colored bg, hover shadow |
| **Checklist** | 8 bullet items with `CheckCircleSolid` (Heroicons/20/solid) on indigo-50 background |
| **CTA** | Dark `bg-gray-900` section with dual-layer gradient blobs, white headline, gray-300 subtext, white-bg CTA button + ghost border secondary |
| **Footer** | Brand column + 3 link columns (Product/Platform/Compliance) + copyright bar with Privacy/Terms/Security links |
