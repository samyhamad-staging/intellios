### CC-7 Fix: Webhook Secrets Encrypted at Rest
- **New file**: `src/lib/crypto/encrypt.ts` — AES-256-GCM encryption module.
  - `encryptSecret(plaintext)` → `enc:v1:<iv>:<ciphertext>:<tag>` format
  - `decryptSecret(stored)` → handles both encrypted and legacy plaintext
  - `isEncrypted(stored)` → prefix check
  - Requires `SECRETS_ENCRYPTION_KEY` env variable (64 hex chars = 32 bytes)
  - Graceful fallback: without key, secrets stored as plaintext with warning
- **Modified files**:
  - `admin/webhooks/route.ts` POST → encrypts before INSERT, returns plaintext in response
  - `admin/webhooks/[id]/rotate-secret/route.ts` → encrypts before UPDATE
  - `lib/webhooks/dispatch.ts` → decrypts before HMAC computation
  - `admin/webhooks/[id]/test/route.ts` → decrypts before test delivery
