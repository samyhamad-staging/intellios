## Decisions

- **P1-334 PDF approach**: Used `window.print()` on a new HTML window rather than a PDF library. Zero new dependencies; browser handles PDF export natively via print dialog. Works cross-platform and produces well-formatted output from the styled HTML.
- **P1-67/68 scope**: Company name field was already present (`companyName`). P1 was satisfied by (a) relabelling to "Enterprise name" and (b) full dark branding matching login page.
- **P1-542 test mechanism**: True OIDC round-trip test by opening NextAuth's `/api/auth/signin/oidc` in a popup. More honest than a mock ‚Äî it actually exercises the configured issuer. Result is inferred from popup close (can't inspect cross-origin content) with an explanatory message to the user.
