## Decisions Made

- Retry on all error types (not just 429/5xx). Rationale: overhead is 7 s max before fallback; simplicity outweighs granularity. Revisit if retry storms emerge.
- Default timeout 120 s (2× the observed worst-case generation time of 60 s).
- Existing graceful fallbacks in `remediate.ts`, `classify.ts`, `orchestrator.ts` are preserved — the resilience layer adds 3 API attempts before those fallbacks trigger.
- No model fallback chain (Sonnet → Haiku on failure). Rationale: Haiku cannot substitute for Sonnet in generation quality; failing clearly is better than silently downgrading output.
