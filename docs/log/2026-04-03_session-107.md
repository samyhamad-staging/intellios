## Decisions

- **P1-81 role detection**: Used `useSession()` client-side rather than a server fetch, matching the existing pattern in the app. The welcome page is already `"use client"` so this is zero-cost.
- **P1-204 action icons**: Used Unicode characters (✦ ▷ ↓ ⚡) instead of lucide icons to avoid adding an import to a page that had no lucide imports. Keeps the bundle lean.
- **P1-433 PagerDuty masked input**: Integration key is sensitive. Used `type="password"` with `autocomplete="new-password"` to prevent browser autofill from leaking the key elsewhere.
