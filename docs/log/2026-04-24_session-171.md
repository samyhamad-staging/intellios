## Step 4 — FFmpeg Composition Reference

`scripts/demo-video/ffmpeg-compose.sh` — reference-only bash script for Session 172 composition. `bash -n` syntax check passed.

Pipeline:

1. Pre-flight — verify inputs exist and narration duration is in the 8:00–9:30 window (soft gate).
2. SVG → PNG (prefer `rsvg-convert`; fallback to FFmpeg).
3. PNG + silence → 3-second title-card MP4 clips (at `fps=30`, `1920x1080`, AAC silence to match concat audio format).
4. Transcode `walkthrough.webm` (VP8 + no audio) → H.264 MP4 with AAC silence so concat can stream-copy.
5. Demuxer concat: cold-open + walkthrough + outro.
6. Overlay `narration.mp3` with `-itsoffset 1.5` so narration starts after the cold-open title card.
7. Final re-encode — `libx264 -crf 23 -preset medium`, `aac -b:a 192k`, `+faststart`.

Output: `scripts/demo-video/output/demo-v1.mp4` — 1080p H.264, YouTube-unlisted-ready.
