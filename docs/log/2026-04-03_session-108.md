## Decisions

- **P2-324 "Policies" breadcrumb node links to /governance**: The governance page is the policy list. There is no separate `/governance/policies` URL. Linking both crumb nodes to `/governance` is accurate.
- **P2-113 no message copy**: Duplicating chat messages would cause the LLM to see prior assistant turns out of context, leading to confused responses. Fresh chat on a pre-populated payload is a better UX.
- **P2-435 inline SVG for copy/check icons**: `production-dashboard.tsx` already uses inline SVG for its custom activity icon. Added the copy and checkmark icons the same way to keep the import list unchanged.
- **P2-390 native `title` tooltip**: Browser-native tooltips are immediate and work with no JS ŌĆö sufficient for a hover legend on filter pills. A custom tooltip library would add complexity with no measurable UX gain at this scale.
