# Session 070 — 2026-03-30 (morning)

## Objective

Two independent improvements: fix a production deployment constraint (Vercel Hobby plan cron frequency limit) and evolve the visual design language to better signal enterprise credibility and AI-native product positioning.

## Actions

### Cron Frequency Downgrade (`vercel.json`)

The Vercel Hobby plan enforces a hard limit of once-per-day for all cron jobs. Two crons were set to sub-daily intervals and were silently failing or being rejected:

- `alert-check`: `*/15 * * * *` (every 15 minutes) → `0 9 * * *` (daily 9am UTC)
- `telemetry-sync`: `0 * * * *` (every hour) → `0 10 * * *` (daily 10am UTC)

The original high-frequency schedules remain the intent for production. Upgrade path: restore `*/15` and `0 *` intervals when upgrading to Vercel Pro.

### Design System v1.2 — Color Palette Evolution

**Primary color shift:** violet-700 (`#6d28d9`) → indigo-600 (`#4f46e5`). The new primary reads as cooler and more enterprise-native — indigo sits closer to the blue end of the spectrum that enterprise buyers associate with trust and stability, while retaining the distinctiveness of a non-commodity palette. Violet-700 read as "creative tool"; indigo-600 reads as "serious platform."

**Files modified:** `src/app/globals.css`, `src/app/landing/page.tsx`, `src/app/login/page.tsx`, `src/app/page.tsx`, `src/components/nav/sidebar.tsx`

**Specific changes:**

- **`globals.css`** — CSS custom properties updated throughout: `--primary` → indigo; surfaces and borders cool-shifted to blue-tinted slate; shadows changed from `rgba(0,0,0,...)` to indigo-tinted rgba for ambient depth; new `.btn-primary` utility class with gradient background (`indigo-600 → violet-600`) and glow box-shadow
- **`login/page.tsx`** — Background changed from flat gray to dark gradient; login card converted to glass-morphism style (backdrop-blur, semi-transparent surface, subtle border)
- **`landing/page.tsx`** — All violet accent references → indigo; CTA buttons use new `.btn-primary` utility
- **`sidebar.tsx`** — Header section receives subtle gradient; active nav item shows indigo glow ring; accent tokens updated to pull from new CSS custom properties
- **`page.tsx`** — Inline color references updated to match new token values
