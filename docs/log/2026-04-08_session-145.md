# Session 145 — 2026-04-08

**Contrast Token Cleanup — WCAG AA Text Scale Shift**

## Context

The accessibility audit (Session 139, findings A-01 and A-03) identified that `--color-text-tertiary` (#94a3b8, slate-400) fails WCAG AA 4.5:1 contrast for normal-sized text. The token appears 723 times across 103 files. Rather than doing a targeted swap of 723 class references, the decision was to darken the global CSS custom property — and shift the entire text scale down one Tailwind step to preserve visual hierarchy.

## Actions

### 1. Calculated WCAG-compliant token values

Wrote a Python script to compute contrast ratios for the full slate palette against white (#ffffff) and dark (#0f172a) surfaces. Found that:
- Simply darkening tertiary to pass 4.5:1 would make it nearly identical to secondary (1.06:1 gap)
- Shifting the entire scale one step solves both tertiary AND secondary contrast while preserving 1.59:1 visual gap

### 2. Updated globals.css — 4 token values + 2 shadcn compat aliases

Light mode:
- `--color-text-secondary`: #64748b → #475569 (slate-500 → slate-600, 7.58:1)
- `--color-text-tertiary`: #94a3b8 → #64748b (slate-400 → slate-500, 4.76:1)

Dark mode:
- `--color-text-secondary`: #94a3b8 → #cbd5e1 (slate-400 → slate-300, 12.02:1)
- `--color-text-tertiary`: #64748b → #94a3b8 (slate-500 → slate-400, 6.96:1)

shadcn compat:
- `--muted-foreground` (light): #64748b → #475569
- `--muted-foreground` (dark): #94a3b8 → #cbd5e1

### 3. Updated chart-tokens.ts

- `chartColors.gray`: #94a3b8 → #64748b (matches new tertiary)
- `chartTextColor`: #64748b → #475569 (matches new secondary)

### 4. Updated documentation

- `docs/design-tokens.md`: Updated hex values and added WCAG ratios in text token table and chart tokens table
- `docs/specs/table-toolbar.md`: Updated Tailwind step references (slate-400 → slate-500, slate-500 → slate-600)
- `docs/audits/a11y-audit-2026-04-07.md`: Marked A-01 and A-03 as RESOLVED, updated remediation plan
- `docs/decisions/014-text-contrast-token-shift.md`: ADR documenting the scale shift decision

### 5. Preserved non-text usages

Left untouched:
- Scrollbar thumb colors (decorative UI chrome, not text)
- Score ring stroke color in `domain-progress-strip.tsx` (decorative SVG)
- Policy audit text token (independent semantic color)

## Summary

Resolved WCAG AA compliance findings A-01 and A-03 by shifting the text color scale one Tailwind step darker (light mode) / lighter (dark mode). 4 CSS custom property changes + 2 shadcn compat alias updates + 2 chart token updates. Zero class-name changes — all 1,380 usages of secondary and tertiary tokens inherit the new values automatically. ADR-014 documents the decision.
