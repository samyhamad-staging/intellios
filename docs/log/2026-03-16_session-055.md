# Session 055 — 2026-03-16

## Summary

Phase 47: Help Copilot + action-capable navigation + Overview Recent Activity redesign.

**Help Copilot (multi-turn):** Upgraded "Ask Intellios" from one-shot Q&A to a persistent multi-turn AI conversation. Replaced the manual streaming state machine with `useChat()`, `MessageBubble` bubbles, and `/api/help/chat` using the AI SDK UIMessage format. The copilot now remembers the full conversation within a session, proactively suggests follow-up actions, and lets the user clear with one click.

**Action-capable copilot:** Added a `suggest_action` tool to the help chat endpoint. When the model knows where the user should navigate, it calls the tool and the panel renders a clickable violet navigation card. Clicking it routes the user to that page and closes the panel.

**Role-aware suggestions fix:** `SUGGESTED_QUESTIONS["/"]` was admin-only; `FALLBACK_QUESTIONS` was a flat `string[]` so `getSuggestions` always returned admin questions for unmatched roles. Fixed by expanding `"/"` to all 4 roles and making `FALLBACK_QUESTIONS` a `Record<string, string[]>` keyed by role.

**Recent Activity redesign:** Admin Overview rows were flat 5-column layouts with raw email addresses and "—" for nulls. Redesigned to two-line rows: agent name on line 1; `"by [username] · [timeAgo]"` on line 2 (username extracted from email before `@`). Status badge stays right-aligned. Cleaner, more scannable.

**1 new file · 2 rewrites · 1 deleted · 0 DB migrations · 0 new npm dependencies · TypeScript: 0 production errors**
