## Key Decisions

- **Compliance officer gets 4-card grid vs reviewer's 3** — compliance officers' primary work is governance posture, not pipeline velocity. Surfacing Governance Hub and Compliance Posture as top-level cards removes the need to navigate there manually.
- **Empty-queue state preserves the "all clear" feeling** — the green checkmark is kept; context is added below it rather than replacing it. This maintains the positive signal while giving actionable forward guidance.
- **Viewer Explore block before Admin Action Queue** — viewers don't own action items, so the Explore block comes first. The action queue items that follow are still useful for viewers to see (they link to read-only views), but contextual navigation is the higher-value content for them.
- **Compliance Posture card links to `/governance`** — there is no dedicated posture route yet. The label is aspirational; the link is correct (governance hub is where posture data lives). When a dedicated posture tab is added, the href needs updating.
